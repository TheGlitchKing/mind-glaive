# Minimal Template

Lightweight CLAUDE.md structure for small projects, scripts, and learning.

## When to Use

- Starting new small projects
- Learning Claude Code context system
- Simple command-line tools
- Quick prototypes
- Solo developers

## Contents

- `CLAUDE.md` - Core structure (Current Focus, Decisions, Patterns, Issues)
- `rules/example-rule.md` - Demonstrates glob pattern syntax

## Customization

Add project-specific sections to CLAUDE.md as needed:

```markdown
## Your Custom Section

<!-- Your content here -->
```

Create new rules in `rules/` when patterns emerge (3+ occurrences).

## Size

- CLAUDE.md: ~1KB
- Rules: As you add them
- Target: < 10KB total
